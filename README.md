# Team-2D-Archer
Legend of Archer를 모델로 만든 게임 
## 기능
W A S D = 이동

### 진행
마을 => 던전 => 보스

## 역할 분담
랜덤 방 생성 / 맵 디자인 = 최우영

캐릭터 이동 / 공격 = 조지현

적AI / 공격 패턴 = 차광호

스킬 / 업그레이드 = 주세웅

보스전 = 준비중
## Git
[개인] 작업 => [Develop] 확인 => [main] 최종
### 규칙
[Test] - 테스트

[Fix] - 기존 코드 수정, 오류 해결 

[Docs] - 에셋추가, 파일수정

[Feat] - 그 외의 작업
## 코딩 규칙
클래스, 함수 => 파스칼 케이스 (첫글자 대문자, 중간글자 대문자)

변수 => 카멜 케이스 (첫글자 소문자, 중간글자 대문자)

private => 스네이크 케이스 (_이후에 소문자_대문자)
